## **How to Install [Nerd Fonts](https://www.nerdfonts.com/)**

Installing [Nerd Fonts](https://www.nerdfonts.com/) is easy! You can either download the font files directly from the [Nerd Fonts](https://www.nerdfonts.com/) website, or you can install them via the command line.

Here's how to install [Nerd Fonts](https://www.nerdfonts.com/) the command line:

1. Open your terminal
2. Type the following command:
```bash
brew tap homebrew/cask-fonts
```

3. Type the following command: 
```bash
brew install --cask font-<font-name>
```

Replace `<font-name>` with the name of the font you want to install. For example, to install the `"Fira Code"` font, you would type :
```bash
brew install --cask font-fira-code
```

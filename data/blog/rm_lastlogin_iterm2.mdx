When you open iTerm or Terminal on your macOS, you may be greeted with a message that displays the last time you logged in. This prompt can be distracting and unnecessary for some users. Luckily, there is a simple way to remove this prompt using the hushlogin file.

## Step 1
Open iTerm or Terminal on your macOS.

## Step 2
Create a new file called .hushlogin in your home directory by typing the following command in the terminal window:

```bash
touch ~/.hushlogin
```

## Step 3
Logout and login again, the `Last Login` prompt should no longer appear.

Please note that this method works for all shells, not just for Zsh shell, and is specific to iTerm or Terminal on macOS. If you're using a different terminal emulator, the steps may vary.

And there you have it! With these simple steps, you can now remove the `Last Login` prompt from iTerm/Terminal on macOS and have a cleaner and more streamlined terminal experience.

If you ever want to bring back the Last Login prompt, simply delete the .hushlogin file by running the following command:

```bash
rm ~/.hushlogin
```

and logout and login again.

In conclusion, the hushlogin file is a convenient and easy way to remove the Last Login prompt on iTerm or Terminal on macOS, it's a quick and easy solution for those who prefer a cleaner terminal experience.
